## 🛠️ Technologies Used

- ⚛️ **Frontend**: React + Vite
- 🎨 **Styling**: Tailwind CSS + Radix UI
- 🧰 **Tooling**: TypeScript, ESLint, React Query, React Router
- 🗂️ **Version Control**: Git + GitHub
- 🌐 **Hosting**: GitHub Pages
